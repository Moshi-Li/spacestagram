## [Spacestagram](https://moshili95.github.io/spacestagram/)

**[Demo](https://moshili95.github.io/spacestagram/)**

An image browsing website using **[NASA API](https://api.nasa.gov/#apod)**.

### Installation & Start

```
yarn install
```

```
yarn start
```

### Build & Deploy to Github Pages

```
yarn build
```

```
yarn run deploy
```
